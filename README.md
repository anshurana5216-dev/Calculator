# Calculator App

A simple and responsive Calculator Web Application built using HTML, CSS, and JavaScript as a part of the internship task.

## Project Overview

This project performs basic arithmetic operations such as:

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- Decimal Calculations
- Clear Screen (AC)
- Delete Last Character (DEL)

The calculator provides a clean and user-friendly interface for performing quick calculations.

## Features

-> Responsive User Interface

-> Real-Time Input Display

-> Basic Arithmetic Operations

-> Clear All Functionality

-> Delete Last Character Functionality

-> Decimal Number Support

-> Error-Free Calculation Handling

## Technologies Used

- HTML
- CSS
- JavaScript
